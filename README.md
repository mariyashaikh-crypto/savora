# 🍽️ Savora AI — Food Demand & Waste Optimization Platform

Savora AI is an AI-powered web application designed to help restaurants reduce food wastage and optimize daily operations. It analyzes key inputs such as day, weather, and demand patterns to predict food requirements and provide actionable insights.

## 🚀 Features

- 🤖 **AI Demand Prediction** — Estimates daily food requirements  
- 📊 **Industry Insights** — Hotel analytics with logistics and **daily waste (kg/day)**  
- 🌍 **Savora Optiserve** — Connects hotels with nearby NGOs for food donation  
- 🧠 **AI Insights** — Smart recommendations for waste reduction  
- 🎯 **Impact Simulator** — Visualizes potential savings and social impact
